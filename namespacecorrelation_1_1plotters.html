Renders multiple histograms overlaid on a single plot.

Each entry in datasets provides a display label and a pointer to the Histogram data. The first dataset defines the axes and labels; subsequent datasets are drawn on the same grid with distinct colors.

Parameters
datasetsVector of labeled histogram pointers.
partial_keyWhich partial to plot (e.g. "Total", "Si-O").
configOptional plot configuration (theme, size, etc.).
hoverOptional hover interaction info.
Returns
A complete SVG document as std::string.
