Applies kernel smoothing to a data series.

This function convolves the chosen kernel with y to produce a smoothed version. The bin width dx fully characterises the sampling grid; the full grid vector is not required and is not accepted to keep the interface lean.

Parameters
dxUniform bin width of the data (e.g., Δr in Å).
yThe dependent variable values to be smoothed.
sigmaKernel bandwidth (standard deviation for Gaussian), in the same physical units as dx.
typeKernel type to use.
Returns
A vector containing the smoothed data (same length as y).
