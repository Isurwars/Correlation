Retrieves the atomic form factors for a given chemical element.

The array contains 9 coefficients (a1, b1, a2, b2, a3, b3, a4, b4, c) based on the Cromer-Mann equation.

Parameters
symbolThe atomic symbol of the element.
Returns
A 9-element array containing the form factor coefficients.
Exceptions
