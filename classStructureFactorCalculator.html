Detailed Description

Computes the static structure factor S(Q) using plane wave summation over reciprocal lattice vectors for periodic systems.

For each reciprocal lattice vector q, computes: rho(q) = sum_j exp(i q . r_j) S(q) = |rho(q)|^2 / N

Results are averaged over q-vectors with similar magnitudes (|q| shells).
