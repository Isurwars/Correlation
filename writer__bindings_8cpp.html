Detailed Description

Pybind11 bindings for correlation::writers types.

Exposes:

  • BaseWriter (abstract base)
  • CSVWriter
  • write_csv() (free-function convenience wrapper)
  • get_writer() (free function wrapping WriterFactory::instance())
  • list_writers() (free function)

Note: WriterFactory is a move-only singleton — it is not exposed as a Python class. Instead, module-level free functions provide access to its registered writers.
