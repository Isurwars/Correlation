Detailed Description

template<typename T>
class correlation::math::Vector3< T >

A lightweight, constexpr, stack-based 3D vector.

Provides basic linear algebra operations for 3D geometric calculations. Supports SIMD optimization for specific types (e.g., double).

Template Parameters
TThe scalar type stored by the vector (e.g., float, double, int).
