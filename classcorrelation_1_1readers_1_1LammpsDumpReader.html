Detailed Description

Reads one or more frames from a LAMMPS custom dump file.

Supports the common ITEM: ATOMS id type x y z format as well as the element-name variant (element x y z) and scaled / wrapped coordinates (xs ys zs). When the file contains more than one timestep the reader returns every frame, making it suitable for LAMMPS MD trajectories.

Uses memory-mapped I/O for lazy frame loading on large trajectories.
