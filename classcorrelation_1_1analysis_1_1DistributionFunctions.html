Detailed Description

Manages the calculation and storage of various distribution functions.

This class includes methods for calculating:

  • Radial Distribution Functions (RDF) g(r), J(r), G(r)
  • Plane and Angle Distributions (PAD)
  • Validation Autocorrelation Function (VACF)
  • Vibrational Density of States (VDOS)
  • Structure Factor S(Q)
  • X-Ray Diffraction (XRD) patterns

It acts as a container for all analysis results associated with a correlation::core::Cell or correlation::core::Trajectory.
