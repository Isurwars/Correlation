Detailed Description

Orchestrates structural analysis across multiple frames of a trajectory.

This class handles iteration over trajectory frames, providing a mechanism to create frame-specific StructureAnalyzer instances. It also manages high-level analysis settings such as start/end frames and periodic interaction toggles.
