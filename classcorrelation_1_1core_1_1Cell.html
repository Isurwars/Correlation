Detailed Description

Represents the simulation cell (periodic box).

This class handles the lattice vectors, periodic boundary conditions, and stores the atoms contained within the cell.

correlation::core::Cell::Cell ( const std::array< double, 6 > &  params)
explicit

Constructs a Cell from lattice parameters {a, b, c, alpha, beta, gamma}.

Parameters
paramsAn array containing the six lattice parameters:
  • params[0]: a (length of vector a in Angstroms)
  • params[1]: b (length of vector b in Angstroms)
  • params[2]: c (length of vector c in Angstroms)
  • params[3]: alpha (angle between b and c in degrees)
  • params[4]: beta (angle between a and c in degrees)
  • params[5]: gamma (angle between a and b in degrees)
