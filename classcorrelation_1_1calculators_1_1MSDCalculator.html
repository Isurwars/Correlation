Detailed Description

Computes the Mean Squared Displacement (MSD) and the diffusion coefficient via the Einstein relation.

MSD(t) = <|r_i(t0 + t) - r_i(t0)|^2>

Atomic positions are unwrapped using the minimum image convention so that periodic boundary condition crossings are handled correctly. The diffusion coefficient can be extracted from the long-time slope: D = MSD(t) / (6 * t) (3D isotropic)
