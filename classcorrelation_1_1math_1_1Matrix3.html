Detailed Description

template<typename T>
class correlation::math::Matrix3< T >

A lightweight, constexpr, stack-based 3x3 matrix.

Uses column-major storage format (standard for OpenGL and BLAS). This means data is stored as an array of three column vectors.

Template Parameters
TThe scalar type stored by the matrix.
