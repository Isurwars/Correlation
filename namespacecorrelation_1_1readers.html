static BaseReader * correlation::readers::findReaderForFile ( const std::string &  filename)
static

Finds a reader for the given filename.

Tries the file extension first via ReaderFactory. When the extension is empty (extensionless VASP files such as POSCAR, CONTCAR, XDATCAR) falls back to a case-insensitive basename check.

Parameters
filenamePath to the file.
Returns
Pointer to the matching reader, or nullptr if none found.
