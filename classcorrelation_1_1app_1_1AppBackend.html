Detailed Description

The main backend class for the application.

This class orchestrates the loading of files, setting of options, running of analyses, and writing of results. It acts as the bridge between the UI/Controller and the core data/analysis logic.

◆ run_analysis()

std::string correlation::app::AppBackend::run_analysis ( )

Runs the analysis based on the current options and loaded trajectory.

This method performs the following:

  1. Sets up bond cutoffs and time steps.
  2. Initializes the TrajectoryAnalyzer.
  3. Computes the mean Distribution Functions (RDF, ADF, etc.).
  4. Calculates VACF and VDOS if applicable.
Returns
A string containing an error message if the analysis failed, or an empty string if successful.
